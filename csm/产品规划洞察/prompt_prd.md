# PRD｜AI-原生客户增长平台（CSM 新产品）

---

## 执行摘要（What / Why）
我们将推出一款 AI-原生客户增长平台，以“Cost-Aware（成本感知） + Agentic AI（智能体） + 合规内生”为差异化核心，帮助 SaaS 公司在更低的每美元留存成本（CbR）下，获得更高的净收入留存（NRR）与更大覆盖率（Accounts per CSM）。该方向与 CSM 市场由“AI 副驾驶 → 多智能体 → 自执行”的代际演进一致，并将合规要求（EU AI Act / Data Act / CPPA ADMT）从约束转化为信任与壁垒。

---

## 产品愿景（Vision）
在 36 个月内，把产品打造成“全球领先的 AI-原生客户增长平台”，通过智能体 AI 从辅助走向自执行，在合规与成本约束下编排端到端客户旅程，实现可预测、可扩展、可持续的收入增长。  
市场趋势：智能体 AI 兴起、CS 收入所有权常态化、数字 CS 规模化与治理趋严。

---

## 目标用户与场景（Who / JTBD）

### 主用户画像
- CSM / AM（中高触达）：在有限工时下覆盖更多账户、降低流失、拿下续约/扩展。  
- CS 主管 / VP CS：可视化团队产出、优化覆盖率与预算、证明 ROI。  
- RevOps / Finance：把 CS 动作与收入、毛利、折扣/让利与成本归因打通。  
- 客户方管理员 / 高管（Self-Serve）：自助获取价值证据（QBR / 业务成效）并配合成功计划。  
- 合规 / 法务 / IT：AI 决策可解释、审计留痕、数据可携与分域治理。

### 关键场景（Top Use Cases）
- 续约与让利一体化：在利润红线与预算内生成谈判方案并一键执行、回写预测。  
- Adoption 最小成本路径：通过教育 / 社区 / In-app / 触达组合实现目标使用率的边际成本最优。  
- Expansion 触发：基于阈值与席位饱和度的扩展信号 → 编排行动 → 收入归因。  
- Churn Rescue：情绪 / 健康下滑触发“限额式”挽救剧本（含高管介入、专项包）。  
- QBR / 价值实现：把用量 → 业务 KPI（省时 / 减少故障 / 合规通过率）可计算并嵌入 QBR。  
- 数字 CS 规模化：低 / 中触达群体由自执行代理托底，经理控盘覆盖率 / 预算 / 输出质量。

## 市场定位（Positioning）
相对 Gainsight、ChurnZero、Totango、Velaris 等竞品：同样具备会议纪要 / 写作 / 健康分 / 编排功能，但强调：
- 将成本维度（CbR / Cost-to-Serve / 让利预算）与 AIGRC（AI 治理 / 合规）作为系统一等公民；  
- 提供在预算与法律约束下落地的自执行（Autopilot）能力。  
细分渗透策略：优先服务成长型 SaaS / PLG + 销售混合客户，在 EU / US 兼容治理要求方面形成强势心智。

## 核心功能（按优先级）
说明：P0 = 必须；P1 = 强力差异化（次阶段）；P2 = 长期护城河/生态

### P0（GA 核心，0–6 个月）
- 统一客户数据模型 2.0（UDM）  
    - 建模：账户 / 席位 / 模块 / 事件 / 情绪 / 成本因子（人力、支持、教育、云、折扣）与收入对象一致建模。  
    - 集成：SFDC / HubSpot、Snowflake、Zendesk、Gmail/Outlook、Slack、Gong/Zoom/Teams。  
    - 验收：主流集成 < 2 小时接通，事件日活 > 100 万条。  
- CSM Copilot v1（嵌入全局）  
    - 功能：会议纪要 / 行动项、邮件 / 消息撰写、上下文问答，一键执行跨工具原子动作。  
    - 验收：>70% 会议自动纪要采用率。  
- Health 2.0 v1：时序 + 可解释健康分（含 SHAP 摘要），与续约/扩展预测联动。  
    - 验收：提前预警时间 ≥ 14 天。  
- CbR 基础度量（成本看板）：人力 / 支持 / 教育 / 云 / 折扣分摊与账户级 Cost-to-Retain。  
    - 验收：经理看板可按细分 / 人群 / 剧本对比单位成本。  
- 低代码编排（基础版）：事件 / 信号触发 → 多渠道动作（邮件 / In-app / 任务 / 会议 / Slack）。  
    - 验收：模板上线 ≤ 1 小时。  
- AIGRC 基础：AI 日志、模型卡、数据分类 / 留痕、可携性导出初版。  
    - 验收：生成“合规包”草稿（EU AI Act / ISO42001 摘要）。

### P1（差异化，6–12 个月）
- Cost-to-Retain Optimizer：针对机会/风险，给出“最小成本达成目标”的行动组合与边际成本、成功率。  
    - 验收：A/B 实证 CbR 下降 ≥ 10%。  
- Agent Builder & 多智能体编排：续约 / Adoption / Expansion / Churn / Compliance 代理可视化编排，带预算 / 让利 / 法律阈值。  
    - 验收：>40% 剧本由 Agent 半/全自动执行。  
- Customer Copilot v1：客户侧自助 QBR 草稿、成功计划、学习 / 社区推荐。  
    - 验收：客户侧月活 ≥ 40%。  
- Value Realization v1：用量 → 业务 KPI 归因与 QBR 自动嵌入。  
    - 验收：>60% QBR 自动生成。  
- BYO LLM / 私有推理选项：企业可选自带模型与专属实例。  
    - 验收：三大主流 LLM 无缝切换。

### P2（护城河与生态，12–24 个月）
- Autopilot（守护式自执行）：在预算、合规与审批阈下，全自动托管低 / 中触达人群。  
    - 验收：目标人群自执行占比 ≥ 70%。  
- 行业套件（FinTech / HealthTech）：指标 / 模板 / 合规清单预置。  
- Agent SDK & Action API 市场：合作伙伴发布领域 Agent 与跨工具原子动作。  
- 本地化 / 主权云与多语言（中 / 日 / 德）：满足数据主权与全球化需求。

## 非功能需求（NFRs）
- 安全与合规：零信任、最小权限执行、审计留痕；AI 评估记录 / 事件报告 / 可解释路径对齐 EU AI Act、CPPA ADMT、ISO/IEC 42001。  
- 可扩展性：百万级事件/日、万级账户实时评分；编排延迟 < 5s。  
- 互操作：CRM / 计费 / 产品内消息 / 支持 / 协作 工具标准连接；Data Act 可携性导出。

## 关键业务指标（KPIs & 12 个月目标）

产品与使用
- 会议纪要自动化采用率 ≥ 70%（P0）  
- 自执行 / 半自执行占比 ≥ 40%（P1）  
- 客户侧 Copilot MAU ≥ 40%（P1）

商业与留存
- NRR +2–3 个百分点；GRR +1–2 个百分点（首年）  
- CbR（单位留存成本）下降 ≥ 10–15%（P1 后）  
- Accounts per CSM +20–30%（首年）

模型与运营
- 健康分提前预警时间 ≥ 14 天；续约预测 MAPE ≤ 15%（目标）  
- 合规完备度：模型卡 / AI 评估记录覆盖率 ≥ 95%；可携性导出成功率 ≈ 100%。

---

## 指标定义（节选）
- CbR：单位留存成本 =（CS 人力 + 支持 + 教育 + 云 + 让利/折扣分摊）/ 保留 ARR。  
- 自执行占比：在预算 / 审批阈下由 Agent 自动完成且通过审核的动作占全部动作比。  
- 提前预警时间：健康分跌破阈值到真实事件（流失 / 拒续）的时间差中位数。

## 范围与不做事项（Scope / Non-Goals）
- In：CS 全生命周期（入职、采用、续约、扩展、挽救）、跨工具编排、成本 / 价值归因、合规治理。  
- Out：完整 CRM 替代、票务系统替代、端到端计费系统；不内置大众化 BI（以导出 / 联邦为主）。

## 体验与交互（UX 概要）
- Manager Console：NRR / GRR / CbR / 覆盖率 / 自执行占比的单页驾驶舱，按分群 / 剧本切片。  
- Playbook & Agent Studio：左侧信号库（使用 / 情绪 / 财务 / 成本 / 意图），右侧预算与合规守护规则；所见即所得回放与回滚。  
- Customer Copilot：客户可查看价值仪表与 QBR 草稿、共同维护成功计划。

## 数据与技术（高层）
- 特征层：时序窗口（7 / 30 / 90 天）、情绪向量、成本因子、席位 / 模块画像。  
- 模型层：序列流失预测（Transformer / 时序森林）；因果 / Uplift（T/DR-learner）用于“行动 → 续约 / 成本”的联合优化；RAG + 函数调用用于 Copilot。  
- 治理层：模型卡 / 偏差监控 / 审计轨迹 / ADMT 披露；可携性导出（Data Act）。

## 关键风险与缓解
- 技术：智能体效果不足 → 演进式交付（先副驾驶后自执行）、多路线试验与人审插桩。  
- 市场采纳：用户对 AI 不信任 → 强化可解释 + ROI 看板 + 守护式 Autopilot 与审批阈值。  
- 合规 / 数据主权：跨区数据 / AI 审计要求高 → AIGRC 内生、主权云 / 本地化、可携性导出。  
- 竞争：巨头快速追赶 → 聚焦细分人群 + 更快迭代 + 生态 / 合作伙伴共建。
